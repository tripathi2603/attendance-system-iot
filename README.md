## 📌 Overview
This project is a smart IoT-based Attendance Management System developed using **Python, RFID, and Web UI (HTML, CSS)**.
It automates attendance tracking using RFID cards and stores data digitally with real-time processing.

## 🛠️ Tech Stack
- **Backend:** Python  
- **Frontend:** HTML, CSS  
- **Hardware:** Arduino, RFID Module  
- **Database:** SQLite / CSV  

## ⚙️ Features
- ✅ RFID-based automatic attendance  
- ✅ Real-time data processing  
- ✅ Clean and simple UI  
- ✅ Fast and accurate tracking  
- ✅ Reduces manual work
